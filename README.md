# DAI 2026

DAI 2026 网站前端项目，基于 Vue 3 + Vite 构建。

## 快速开始

### 前置要求

- Node.js (推荐 18.x 或更高版本)
- npm 或 yarn

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

构建完成后会生成 `dist` 目录，包含生产版本的静态文件。

### 部署

1. 连接远程服务器
2. 打开文件夹：`/www/wwwroot/www.adai.ai/dai/2026`
3. 用 `dist` 目录中的新文件覆盖原有文件
4. 访问 https://www.adai.ai/dai/2026 验证部署是否生效
